Primo esercizio

Stai costruendo un applicazione per gestire una lista di server.

Di seguito le attività da svolgere insieme

  • All'interno di quest'ultimo, inserisci due componenti server
  • Prova a inserire il template di servers inline
  • Prova a colorare il titolo h3 di servers di blu con lo stile css
  • Usa lo stile inline

Di seguito, le attività da provare a svolgere da solo

  • Aggiungi un component header, usa il codice delle navbar di bootstrap per impostare l'html che trovi qui
  • Aggiungi un pulsante "home" che mandi all'homepage della tua azienda.
